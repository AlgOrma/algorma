Merge Overlapping Ranges MEDIUM SOLVING
PROBLEM STATEMENT
Given a collection of intervals [start, end], merge all overlapping intervals and return the non-overlapping set that covers the same span. Intervals touching at an endpoint count as overlapping.
in: [[1,3],[2,6],[8,10],[15,18]]
out: [[1,6],[8,10],[15,18]]
MY APPROACH
Sort by start. Walk left→right keeping the last merged interval; if the next start ≤ last end, extend the end, else push a new one. O(n log n) for the sort, O(n) sweep.
YOUR SOLUTION spoiler-free 
function mergeRanges(intervals) {
  intervals.sort((a, b) => a[0] - b[0]);
  const out = [intervals[0]];
  for (const [s, e] of intervals.slice(1)) {
    const last = out[out.length - 1];
    if (s <= last[1]) last[1] = Math.max(last[1], e);
    else out.push([s, e]);
  }
  return out;
}

Longest Unique Window MEDIUM
last revised 6d ago · revised 3× · created 18d ago
Given a string, find the length of the longest substring without repeating characters. Return the length only.
in: "abcabcbb" → 3 ("abc")
in: "bbbbb" → 1 ("b")
// solve here from memory — your saved code stays hidden▋ 
function lengthOfLongest(s) {
  const seen = new Map();
  let left = 0, best = 0;
  for (let r = 0; r < s.length; r++) {
    if (seen.has(s[r])) left = Math.max(left, seen.get(s[r]) + 1);
    seen.set(s[r], r);
    best = Math.max(best, r - left + 1);
  }
  return best;
}
